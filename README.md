# PriceTag 📊
### Barcode-Based Competitor Price Intelligence for E-Commerce

> Scan a barcode. Know the market price. Get alerted when it changes.

PriceTag is an Android app that turns any barcode into a live competitor price intelligence feed. Built for e-commerce teams and retail buyers who need to monitor market pricing without manual research.

## ✨ What It Does

- **Instant barcode scanning** — point camera at any product barcode to pull pricing data
- **Price history charts** — visualize how competitor prices have changed over time
- **Background price monitoring** — app checks prices automatically even when closed
- **Push notification alerts** — get notified the moment a tracked product price changes
- **Local database** — all tracked products and history stored on-device for fast access

## 🛠 Tech Stack

| Component | Technology |
|---|---|
| Language | Kotlin |

## 🚀 Getting Started

```bash
git clone https://github.com/SadraAlio/PriceTag.git
```

Open in Android Studio, sync Gradle, and run on a physical device or emulator.

**Requirements:**
- Android 8.0+ (API 26+)
- Android Studio Hedgehog or later
- Physical device recommended for barcode scanning
